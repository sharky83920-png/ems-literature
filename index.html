專屬金鑰

金鑰只存在這台裝置的瀏覽器裡，不會上傳到任何地方。沒有金鑰也能查閱文獻，只是不能修改關鍵字或啟動搜索。

第一次使用？點我看金鑰申請懶人包（約 2 分鐘）
  1. 用手機或電腦開啟 github.com/settings/personal-access-tokens/new（要登入你的 GitHub）
  2. Token name 填 ems-literature；Expiration 選最長的
  3. Repository access 選 Only select repositories → 勾選 ems-literature
  4. 展開 Repository permissions：Contents 選 Read and write、Actions 選 Read and write
  5. 按最下方綠色 Generate token → 複製那串文字 → 貼到上面欄位按「儲存金鑰」

立即搜索

不想等每週一？按下去馬上到雲端搜一輪最新文獻，約 3-5 分鐘後重新整理本頁就能看到。

手動收錄文獻

自己找到的文獻，把 PubMed 網址、PMID 或 DOI 貼進來，系統會自動抓齊書目資料歸檔。

關鍵字清單

改完立即儲存，下次搜索（自動或手動）就會用新清單。刪掉關鍵字不會刪除已收錄的文獻。

新增主題群組

這個網站怎麼運作

每週一早上 6 點，雲端機器人會自動用「關鍵字管理」裡的清單去搜 PubMed、Europe PMC、Semantic Scholar 三個資料庫，新文獻自動進到這裡。你的電腦不用開機。

搜尋小技巧

  • 搜尋框輸入中文（標題／摘要翻譯後）或英文都可以，邊打邊過濾
  • 點標籤可以只看某個主題；「本週新進」只顯示最近 8 天新收錄的
  • 摘要點一下可以展開全文

「待翻譯」是什麼？

剛搜進來的文獻是英文的。定期跟 Claude 說一聲：「幫我翻譯文獻庫的新文章」，中文標題與中文摘要就會補上。

想看某篇的全文中文翻譯？

按文章卡片上的「申請全文翻譯」，照跳出來的三步驟做即可。翻譯完成後卡片會出現「看中文全文」按鈕。圖表會保留原圖＋中文圖說，AI 解讀內容會特別標示，引用前請核對原文。

版權說明

「看 PDF」只提供 Open Access（開放取用）文獻的官方連結。付費牆內的文章請透過醫院／學校管道取得，自行取得的 PDF 請勿上傳到本站。
